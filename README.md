# PIC18F45K20_ControlMotor
El proyecto se divide en cuatro etapas:
- Regulación de velocidad en bucle abierto de un motor de CC con un control PWM
- Medida de velocidad del motor de CC mediante un sensor que genere pulsos de frecuencia variable
- Regulación de velocidad en bucle cerrado del motor de CC con un control PI
- Transferencia de información serie asíncrona (UART-BLUETOOTH) para el diseño de un entorno HMI
